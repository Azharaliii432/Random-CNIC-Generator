# Random-CNIC-Generator

A simple web based Random CNIC Generator built using HTML,CSS,JS.
This project generates fake and randomly structured CNIC numbers for learning purpose only.

## Disclaimer
This tool does not generate real CNICs.
All generated CNIC numbers are random and invalid.

## Features
Generates random CNIC numbers in valid Pakistani CNIC format (#####-#######-#).
Lightweight and fast
No backend required
Clean and Simple UI

## Tech Stack
HTML5
CSS3
JavaScript
